About these examples


Created with love by collaboration between Langner.com and NetworkGeekStuff.com

We created these examples to help OT-BASE system users to visualize the JSON exported data themselves since we realize there are many customer specific use-cases that are better served by quickly building the visualization in-house in contrast to any generic visualization inside OT-BASE that cannot always satisfy specific needs or integrate with external systems. However here we are trying to propose a few examples on how to use open source visualization tools like the D3 library to do something intereseting yourself.
